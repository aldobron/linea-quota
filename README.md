# linea-quota
Linea orizzonte su fotocamera per escursionismo.

La linea si tiene orizzontale indipendemente dal rollio/beccheggio ma, poichè il calcolo basato sull'accelerometro dipende dalla specifica fotocamera, è necessario calibrare (una sola volta) l'app con gli appositi pulsanti in modo che la linea resti fissa sull'orizzonte anche inclinando lo smartphone.  
Comunque la maggior precisione si ha con rollio=0 e beccheggio =0, ovvero fotocamera perfettamente verticale. Questo corrisponde alla situazione in cui la linea è sovrapposta alla croce bianca a centro schermo.  

L'app per Android è "LineaQuota-signed.apk".  
Il file html da aprire da browser è "index.html".
