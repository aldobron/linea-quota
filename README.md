# linea-quota
Linea orizzonte su fotocamera per escursionismo
La linea si tiene orizzontale indipendemente dal rollio/beccheggio.
Calibrare con gli appositi pulsanti in modo che la linea resti fissa, poichè il calcolo dipende dalla specifica fotocamera
L'app per Android è "LineaQuota-signed.apk"
Il file html da aprire da browser è "index.html"
